# SafeSend
Helping victims of domestic violence, trafficking, or situations of immediate danger to send a distress signal without arousing the suspicions of the abuser
This is an Android application developed in Kotlin that disguises itself as a regular application (calculator, weather app, or shopping list). The user configures a specific gesture (for example, entering a secret code on the fake calculator or shaking the phone in a certain way) which, without changing the display, triggers a series of secure actions.
